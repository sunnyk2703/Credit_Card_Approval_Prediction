```bash

```

Go to project folder

```bash

```

# 🤖 Machine Learning

The model predicts approval based on:

- Gender
- Married
- Dependents
- Education
- Self Employed
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area
